I am a researcher, working for the U.S. Air Force (out of San Antonio, Texas) on information systems for defense procurement. I enjoy projects that involve a mix of research and software development, especially those involving simulation modeling, machine learning or statistical/econometric methods.

Open Source

  • Go language bindings to the TensorFlow C API. Nightly and release builds of the Go language bindings to the TensorFlow C API, including Go-compiled TensorFlow protocol buffers and generated Go wrappers for TensorFlow operations.
    gh/wamuir/graft
  • Struct and testdata generation for the DoD’s Procurement Data Standard. XSL transform of the Procurement Data Standard (PDS) schema to generate structs. The objective of this project is to build a testsuite of PDS XML files for system testing.
    gh/wamuir/procurement-data-standard
  • Data exchange for sparse and dense matrices. Input and output of matrices in coordinate format (sparse) and array format (dense) via the Matrix Market format defined by NIST. Data can be integers, complex numbers, real numbers or given as a coordinate pattern, with symmetric, skew-symmetric, Hermitian or general symmetry structure. This implementation is for Go; I/O libraries are also available in other languages, including C, Python, Julia, Haskell, etc.
    gh/wamuir/matrix-market
